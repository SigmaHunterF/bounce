# Bounce

A top-down 2D survival shooter made in Godot 4.

## About

You are a lone player surrounded by slimes that chase you down. Survive as long as you can, fight through waves of enemies, and see how far you can go before they get you.

## Gameplay

- Move with arrow keys or WASD
- Aim with the mouse
- Click to shoot
- Kill 10 slimes to advance to the next wave
- Slimes get faster every wave
- You have 3 hearts — don't let the slimes touch you

## Features

- Wave-based enemy spawning
- Day/night cycle that shifts over time
- Heart-based health system
- Score/wave progress bar

## Built With

- [Godot 4](https://godotengine.org/)
- GDScript
